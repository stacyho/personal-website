DumpIt

DumpIt is a web app that seeks to transform common complaints into startup ideas. It provides a platform where individuals can share their complaints and help fund start-ups that pledge to solve these problems.

Built with HTML/CSS, JavaScript, and Handlebars.js for front-end, and Node.js, Express, and MongoDB for backend. Personally worked on front-end development.

Teammate: Sherry Yang

MIT 6.148 Web Programming Competition | January 2016

WordTree

Word Tree is a Unity app designed to help children learn English letter-sound correspondence, sound blending, and sight word recognition. The idea originated from a literacy app prototype created for the Global Literacy Project.

Built for Android tablets using Unity game engine and MonoDevelop.

Personal Robots Group at MIT Media Lab | Summer 2015
