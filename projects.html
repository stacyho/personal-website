Babble

Babble is a live, interactive platform where MIT students are encouraged to have candid conversations and can provide instant support to each other.

Built with ReactJS, Node.js, and MongoDB.

MIT 6.170 Software Studio | Fall 2016

DumpIt

DumpIt is a web app that seeks to transform common complaints into startup ideas. It provides a platform where individuals can share their complaints and help fund start-ups that pledge to solve these problems.

Built with Node.js, Express, and MongoDB.

MIT 6.148 Web Programming Competition | January 2016

WordTree

Word Tree is a Unity app designed to help children learn English letter-sound correspondence, sound blending, and sight word recognition.

Built for Android tablets using Unity game engine.

MIT Media Lab's Personal Robots Group | Summer 2015
